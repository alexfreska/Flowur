I have these functions working, they use extremely sloppy SQL techniques but I wanted to get them working and then go back to revise them. 
______________________________________________________________________________
DATABASE


connect()
        Opens connection to database.


disconnect($connection)
        Closes an open connection.


______________________________________________________________________________
GET LIST


getUserCharts($userId)
        Gets the users charts and lists their titles.


getUserPubs($userId)
        Gets the users publications and lists the chart title and template.


getTemplates()
        Gets all existing templates and lists the title and info.


getTemplatesForAChart($userId,$chartId)
        Gets all existing templates and lists the title and info aswell as a status indicator of whether the chart has already been published with the template. 
[NOTE: If the chart is already published a perma-link should be included for easy access.]


______________________________________________________________________________
ADDS


addUser($username,$password,$firstName,$lastName,$email)
        Adds a user account unless the username or email is already in use.


addChart($title,$data,$userId) 
        Adds a chart to the database unless the user already has one with the same title.
[NOTE: function does not test for a non existent userId number]


publishChart($userId,$templateId,$chartId)
        Creates a publication instance unless an identical one already exists.
______________________________________________________________________________
UTILITY


getTemplateInfo($templateId)
        Returns an array containing template title and info for templateId.


getTemplateFile($templateId)
        Returns the template file associated with templateId.


getChartTitle($chartId)
        Returns the title associated with chartId.


getChartData($chartId)
        Returns the chart data associated with chartId.
______________________________________________________________________________
CHECKS


alreadyPublished($list,$templateId)
        Checks a list that contains template ids for templateId, and returns 1 if it is found.


conflictingChartTitle($title,$userId)
        Checks if user associated with userId already has a chart with the title.


userExists($userId)
        Checks if there is a user associated with userId.
